link-list-cycle

Controls:

+ Add Node
- Remove Node
Right Move forward one iteration
Left Move back one iteration
Up Move cycle up one node
Down Move cycle down one node
r Reset the iterations and indexes

This is a an illustration of how to detect a cycle in a linked list by iterating through the list where one visitor moves 1 node at a time and a second vistor moves 2 nodes at a time. The cycle is detected when both vistors have the same index.
